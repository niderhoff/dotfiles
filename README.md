# linux.dotfiles

## Preqrequisites

    $ pip install dotfiles

## Installation

    $ cd
    $ git clone https://github.com/niderhoff/linux.dotfiles.git ~/.dotfiles
    $ cd ~/.dotfiles
    $ sudo make linux
